## Cambiar margen en prims.css si se desea controlar el bloque con auto
```css
/* Code blocks */
pre[class*="language-"] {
	border-radius: .5em;
	border: .3em solid hsl(0, 0%, 33%); /* #282A2B */
	box-shadow: 1px 1px .5em black inset;
	margin: 0 auto;
	overflow: auto;
	padding: 1em;
}
```
